# Map Scraper

A Google Maps scraping utility powered by **Playwright** and **Gemini API** to extract business leads such as:

- Company name
- Address
- Phone number
- Website
- Email (if available)

Leads are saved in:

- JSON (`data/map_search_leads.json`)
- CSV (`data/map_search_leads.csv`)
- MongoDB (`leadgen.map_leads` collection)
